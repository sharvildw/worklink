# WorkLink 💼

WorkLink is a job marketplace web application where users can post jobs and workers can apply for them.  
This project includes frontend pages and a Node.js backend.

---

## 🚀 Features

- User login / register
- Worker dashboard
- User dashboard
- Job marketplace
- Job details page
- Admin panel
- Backend API using Node.js
- MongoDB database connection

## ⚙️ Installation

### 1. Clone repo


git clone https://github.com/sharvildw/worklink.git

cd worklink


### 2. Install backend packages


cd backend
npm install


### 3. Create .env file


PORT=5000
MONGO_URI=your_mongodb_link
JWT_SECRET=secret


### 4. Start server


nodemon server.js


or


node server.js
